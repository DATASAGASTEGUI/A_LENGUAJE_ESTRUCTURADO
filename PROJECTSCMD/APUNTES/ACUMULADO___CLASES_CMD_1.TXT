(*) PARAMETROS PARA CAMBIAR EL PROMPT DEL SISTEMA

    $p Mostrar el trayecto actual  
    $g Mostrar el separador >  
    $l Mostrar el separador <  
    $b Mostrar el separador |  
    $q Mostrar el separador =  
    $$ Mostrar el separador $  
    $t Mostrar la hora  
    $d Mostrar la fecha  
    $v Mostrar la versión del sistema  
    $n Mostrar la unidad actual
    $h Retroceso. Borra el caracter previo  
    $e Escape. Muestra el caracter \  
    $_ Retorno de carro y salto de línea (equivale a chr 10 y chr 13 ascii    

(*) CAMBIAR EL PROMPT PARA QUE MUESTRE LA FECHA ACTUAL DEL SISTEMA Y QUE TERMINE EN ">"

***   COMANDO : COPY   ***
**************************

(*) MOSTRAR LA AYUDA DEL COMANDO

    COPY /?

(*) CREAR UN ARCHIVO CON INFORMACION QUE INGRESARA DESDE EL TECLADO. EL ARCHIVO SERA SIN FORMATO (CODIGO ASCII) QUE LLAMARA HOLA.TXT

    COPY CON HOLA.TXT
    ESTO QUE ESCRIBO CONTENDRA EL ARCHIVO(CTRL+Z)

(*) VISUALIZAR EN PANTALLA EL CONTENIDO DEL ARCHIVO

    TYPE HOLA.TXT

(*) COPIAR EL CONTENIDO DE TODOS LOS ARCHIVO CON EXTENSION .TXT EN OTRO ARCHIVO QUE LLAMARA ALMACEN.TXT

    (**) CREAREMOS PRIMERO 2 ARCHIVOS CON EL COMANDO COPY QUE LLAMAREMOS A.TXT Y B.TXT. NOTAR QUE CTRL+Z CIERRA Y GRABA EL ARCHIVO

         COPY CON A.TXT
         ESTE ES MI PRIMER ARCHIVO(CTRL+Z)(ENTER)

         COPY CON B.TXT
         ESTE ES MI SEGUNDO ARCHIVO(CTRL+Z)

    (**) AHORA COPIAREMOS DICHOS ARCHIVOS EN EL ARCHIVO "ALMACEN.TXT", PARA LO CUAL PODEMOS HACERLO DE 2 FORMAS DISTINTAS

         COPY *.TXT ALMACEN.TXT
         COPY A.TXT+B.TXT ALMACEN.TXT

    (**) VISUALIZAR EL CONTENIDO DEL ARCHIVO "ALMACE.TXT"

         TYPE ALMACEN.TXT

(*) COPIAR EN UN ARCHIVO EL RESULTADO DEL COMANDO "HELP". EL ARCHIVO LLAMARA "RESUMEN.TXT" Y ESTARA EN LA RUTA "C:\ALUMNOUTP"

    HELP > C:\ALUMNOUTP\RESUMEN.TXT      (CREAR UN ARCHIVO)

(*) AÑADIR EN EL ARCHIVO "RESUMENT.TXT" DE LA CONSULTA ANTERIOR LA AYUDA DE LA SINTAXIS DEL COMANDO "DATE"

    DATE /? >> C:\ALUMNOUTP\RESUMEN.TXT  (AÑADIR EN UN ARCHIVO)
	
(*) CREAR UN ARCHIVO EN BLANCO

    TYPE NUL > ARCHIVO.TXT

***********************************************************
***   CONCATENAR EN UNA MISMA LINEA A VARIOS COMANDOS   ***
***********************************************************

(*) MOSTRAR LA FECHA Y HORA DEL SISTEMA SIMULTANEAMENTE EN EL SIGUIENTE FORMATO 08/07/2008  10:21 a.m.

    DATE /T & TIME /T
    CLS & DATE /T & TIME /T

(*) INDICAR QUE SE GRABA EN EL ARCHIVO "SALIDA.TXT" DE LA CONSULTA SIGUIENTE "CLS & DATE /T & TIME /T > SALIDA.TXT"

    SOLO EL CONTENIDO DE LA HORA

(*) INDICAR QUE SE GRABA EN EL ARCHIVO "SALIDA.TXT" DE LA CONSULTA SIGUIENTE "CLS & DATE /T > SALIDA.TXT & TIME /T >> SALIDA.TXT"

    PRIMERO SE CREA EL ARCHIVO SALIDA.TXT CONTENIENDO LA FECHA
    LUEGO SE AÑADE LA HORA EN EL ARCHIVO "SALIDA.TXT" QUE YA ESTA CREADO ANTERIORMENTE.
    NOTAR QUE EL AÑADIR ES VALIDO SIEMPRE Y CUANDO EL ARCHIVO YA EXISTE PREVIAMENTE.

***************************************************************
***   FILTROS, REDIRECCIONAMIENTO, BUSCAR, ORDENAR, PAUSA   ***
***************************************************************

(*) SI UN ARCHIVO ES MUY EXTENSO COMO VERLO CON PAUSA CADA 23 LINEAS

    TYPE ARCHIVO.TXT | MORE

(*) ORDENAR UN ARCHIVO POR FILAS ALFABETICAMENTE CRECIENTE

    TYPE ARCHIVO.TXT | SORT

(*) ORDENAR UN ARCHIVO POR FILAS ALFABETICAMENTE DECRECIENTE

    TYPE ARCHIVO.TXT | SORT /R

(*) BUSCAR UNA CADENA DIFERENCIANDO MAYUSCULAS DE MINUSCULAS

    FIND "CASA" ARCHIVO.TXT

(*) BUSCAR UNA CADENA NO DIFERENCIANDO MAYUSCULAS DE MINUSCULAS

    FIND "CASa" ARCHIVO.TXT /I

(*) COPIAR EL CONTENIDO DE UN ARCHIVO EN OTRO. LUEGO AÑADIR EN DICHO ARCHIVO OTRO

    TYPE ARCHIVO1.TXT >  ARCHIVO.TXT
    TYPE ARCHIVO2.TXT >> ARCHIVO.TXT

(*) COPIAR EN UN ARCHIVO UN LISTADO SIMPLE

    DIR > LISTADO.TXT

(*) MOSTRAR UN LISTADO DE TODO EL CONTENIDO DEL DISCO DURO CON PAUSA

    DIR /S | MORE

(*) VISUALIZAR UN FICHERO DE TEXTO CON PAUSA

    MORE < ARCHIVO.TXT

(*) ORDENAR UN FICHERO ALFABETICAMENTE EN PANTALLA

    SORT < ARCHIVO.TXT

(*) ORDENAR UN FICHERO ALFABETICAMENTE A PARTIR DE UNA COLUMNA

    SORT /+1 < DATOS.TXT
    SORT /+2 < DATOS.TXT

(*) ENCONTRAR LAS FILAS QUE CONTIENEN UNA CADENA DE CARACTERES

    FIND "CASA" < ARCHIVO.TXT

(*) ENVIAR EN CONTENIDO DE UN ARCHIVO A LA IMPRESORA

    TYPE ARCHIVO.TXT > PRN

(*) IMPRIMIR LA ETIQUETA Y EL NUMERO DE SERIE EN IMPRESORA

    VOL > PRN

(*) CREAR EL FICHERO VERSION.TXT CONTENIENDO LA VERSION ACTUAL DE MS-DOS

    VER > VERSION.TXT

(*) CREAR EL FICHERO DIRECTORIO.TXT CON EL DIRECTORIO RAIZ DE C:

    DIR C:\ > DIRECTORIO.TXT

(*) ORDENAR UN TEXTO INTRODUCIDO POR TECLADO EN UN FICHERO

    SORT > AMIGOS.TXT (ENTRADA TECLADO Y SALIDA FICHERO AMIGOS.TXT)

(*) ORDENAR UN FICHERO EN OTRO

    SORT < AMIGOS.TXT > AMIGOS1.TXT /R (DECRECIENTE)

(*) ORDENAR UN FICHERO EN LA IMPRESORA

    SORT < AMIGOS.TXT > PRN (ENTRADA FICHERO Y SALIDA IMPRESORA)

(*) BUSCAR UNA CADENA EN UN FICHERO Y CREAR OTRO CON EL RESULTADO DE LA BUSQUEDA

    FIND "ARTURO" < AMIGOS.TXT > RESUMEN.TXT (ENTRADA FICHERO SALIDA FICHERO)

(*) AÑADIR LA AYUDA DE LA ORDEN TIME A UN FICHERO

    TIME /? > ARCHIVO.TXT

(*) TOMANDO EL ARCHIVO ANTERIOR AÑADIR AL FINAL LA AYUDA DE LA ORDEN DATE

    DATE /? >> ARCHIVO.TXT

(*) VISUALIZAR EL CONTENIDO DEL ARCHIVO ANTERIOR

    TYPE ARCHIVO.TXT

(*) GRABAR EN UN ARCHIVO EL ARBOL DE DIRECTORIOS DE LA RAIZ DE LA UNIDAD C:

    TREEE > ARCHIVO.TXT 

(*) MOSTRAR CON PAUSA EL ARBOL DE DIRECTORIOS DE LA RAIZ DE LA UNIDAD C:

    TREEE | MORE

(*) MOSTRAR LOS BYTES LIBRES DE LA UNIDAD C:

    DIR | FIND "BYTES LIBRES" /I
    DIR | FIND "bytes libres"
